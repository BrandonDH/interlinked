"Interlinked"

This is a collection of Design and Engineering related links.

I've visualized each URL here using an algorithm to process the URL as a string, and generate an SVG artifact.

As I go about building things throughout my day, I often come across useful resource on the internet. I don't want to lose these links. The habit of bookmarking has never been easy to develop for me.
Right now the idea of visualizing my collection with svg seems fitting. I did end up using chatgpt for some parts of the algorithm, simply because I was more interested in the outcome than the code; like parcing the url string. But now that I have the foundation, I think I'll go back into the JavaScript and fine tune things.
